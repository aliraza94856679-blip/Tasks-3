# 📖 Overview

This repository contains **20 Python practice programs** focused on decision-making concepts.

The purpose of this project is to strengthen problem-solving skills by implementing different conditional statements in Python. Each program demonstrates a real-world use case, helping beginners understand how program execution changes based on different conditions.
